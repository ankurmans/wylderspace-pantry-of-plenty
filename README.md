# Wylder Space - Pantry of Plenty Sales Page

A world-class standalone sales landing page built with Next.js, TypeScript, and Tailwind CSS, integrated with Figma MCP for automated design token extraction.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js App Router pages and layouts
- `components/` - React components
  - `sections/` - Sales page section components
  - `ui/` - Reusable UI components
  - `layout/` - Layout components
- `lib/` - Utility functions and integrations
- `styles/` - Global styles and design tokens
- `public/` - Static assets

## Figma Integration

This project integrates with Figma MCP for automated design token extraction. Design tokens are synced from Figma and converted to CSS variables.

## Deployment

This project is configured for deployment on Vercel. Simply push to your repository and Vercel will automatically deploy.

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Figma MCP Integration
